#Projeto: Análise de Lucratividade - Caseirinhos da Pri#

###O que você fez?###
Bom, no SQL eu organizei a base de dados bruto;
no EXCEL, limpei os dados nulos e fiz o cálculo do Lucro Real por produto;
e por último no POWER BI, fiz o Dashboard com faturamento total, descobri qual o bolo mais vendido e qual produtos não podem faltar na minha confeitaria atesanal.

###Resultado###
o Dashboard mostra um faturamento de 435 e destaca o bolo de baunilha 2kg como o campeão de vendas.
